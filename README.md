## Overview
This project fetches, processes, and presents **Indian MCA (Ministry of Corporate Affairs) financial data** using a Python-based data pipeline and a lightweight Flask web application.

The objective is to build a reusable **data plumbing layer** that can later be extended to:
- Additional data providers
- APIs
- ESG / compliance use cases
- Scalable deployments
